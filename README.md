  # SQL-DataWarehouseProject
  A data warehouse project using sql server icluding ETL ,data modelling and analysis

**Project Overview: Modern Data Warehouse & Analytics Platform**

**1. Data Architecture**

  Designed a **modern data warehouse** using the **Medallion Architecture**, organizing data into three layers:

  * **Bronze Layer** – Raw ingestion of source data with minimal transformation.
  * **Silver Layer** – Cleaned and standardized data with applied transformations and schema enforcement.
  * **Gold Layer** – Curated, business-ready datasets optimized for analytics and reporting.

**2. ETL Pipelines**

Developed automated **ETL pipelines** to:

* Extract data from multiple source systems
* Transform and clean data (handling nulls, duplicates, data types, and business logic)
* Load processed datasets into the warehouse layers

**3. Data Modeling**

Designed analytical data models including:

* **Fact tables** for transactional and measurable metrics
* **Dimension tables** for descriptive attributes
* Implemented **Star Schema** principles to optimize query performance and reporting.

**4. Analytics & Reporting**

Built SQL-based analytical queries and dashboards to:

* Generate business insights
* Track KPIs and trends
* Support decision-making through interactive reports.
